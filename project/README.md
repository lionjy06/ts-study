최종 프로젝트 폴더입니다

## JS 프로젝트에 타입스크립트 적용하기

1. 타입스크립트 프로젝트 생성 (이미있음)
    - NPM 초기화
    - 타입스크립트 라이브러리 설치
    - 타입스크립트 설정

2. 명시적인 'any' 선언하기
    - 'tsconfig.json' 파일에 'noImplicitAny' 값을 'true'를 넣어주기

3. 프로젝트 환경구성
    - babel, eslint, prettier
4. 외부 라이브러리 모듈화 => index.d.ts를 찾아가 모듈의 타입을 저징해준 파일을 가져온다.

5. strict 옵션추가후 타입 정의
